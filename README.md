# 声笔输入法主页

## 目录结构

- `/.vitepress/`: VitePress 生成器配置
- `/docs/`: 教程文件
- `/posts/`: 博客文章
- `/pages/`: 文章的分类和标签
- `/components/`: Markdown 中用到的 Vue 组件
- `/src/`: 组件引用的 JavaScript 类和函数
- `/public/`: 静态资源文件
- `/sbtf/`: 声笔条反，在线练习声笔飞系的部首

## 开发

### 安装依赖

```bash
pnpm i
```

### 预览

```bash
pnpm start
```

### 构建

```bash
pnpm run build
```
