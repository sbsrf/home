# 4 测评

## 4.1 测评指标
-  综合指标：在一个人的输入法生命周期中的有效输入量，也就是在这段时间内对输入速度的积分，也可等价地用长期、有效的平均速度来衡量，类似 CDF 和 PDF 的关系。有效输入量不包含仅仅为了学习而进行的练习，练习是代价和成本而不是收益。同时，还要扣除使用中的代价，如更快的击键、更强的人机交互等。
-  简单高效：简单包含易学（含易记或难忘）、易用；高效包含快速、准确。输入法的根本目标就是简单高效地输入文字。
-  快速性（高效性、有效性）：码长（信息熵）、击键（信道容量）、回改、选择；单位时间内输入的字、词数量；瞬时速度与平均速度；就是虚拟输入信道的容量。
  - $C=\lim_{T\to\infty}\frac{\log N(T)}{T}$
-  准确性（可靠性）：少错误
-  简单性：易学、难忘
- 持久性：负荷（脑、眼、手）轻，也就是易用
- 规范性：符合语文知识、符合国家标准、少歧义，与简单性重叠较多
- 适应性：可以逐渐适配用户的使用习惯，主要是字词频率的调整
- 渐进性：门槛低、上限高
- 经济性：对软硬件的要求低
- 无扰性：不作恶
- 易达性：容易找到（预装、推送）、下载、安装、配置
- 顺畅性：是不是会有卡顿？
- 重码率？越低越好吗？
- 盲打率？越高越好吗？
- 手感？究竟是什么？当量？互击？跨行？错手？
- 字词量？字词的收录量越大越好吗？
- 流行性？知名度高（广告、软文、介绍、讨论、赛文）、用户量大
